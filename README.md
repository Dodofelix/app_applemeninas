# Apple Meninas — Gestão de Vendas

Sistema web (React + Vite + Firebase) para pedidos, clientes, painel administrativo e contratos.

## Desenvolvimento

Requisitos: Node.js e npm.

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Saída em `dist/` (deploy em Vercel, Netlify ou similar).

## Variáveis de ambiente

Configure as chaves do Firebase conforme o projeto (ex.: `.env` com prefixo `VITE_`).
